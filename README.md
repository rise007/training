ITトレーニング用
====

身内のIT基礎力底上げ用のトレーニングリポジトリ

## Description
このリポジトリは身内のIT基礎力（特にプログラミングに関して）を底上げする目的で
作成しています。
参加者はディレクトリを切ってソースコードを入れていきましょう。

## Requirement
特になし

## Usage
1. リポジトリにソースをプルリク
2. 僕がレビュー
3. なおしてマージ

git clone <このリポジトリ>
